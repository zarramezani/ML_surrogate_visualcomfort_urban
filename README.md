# Abstract

Visual comfort is a key dimension of environmental quality in dense urban blocks, yet multi-metric assessment via climate-based simulation is computationally demanding. This study develops a data-driven framework to predict façade-level performance classes for six visual comfort metrics. Using Grasshopper with Honeybee/Ladybug, 2,000 parametric urban-block scenarios were generated for the climate of Tehran. On façade sensor grids, Vertical Sky Component (VSC), Vertical Daylight Factor (VDF), Sunlight Autonomy (SA), Direct Sun Hours (DSH on 21 March), Peak Irradiance (Peak-Irr), and Visibility Percent (VP; a proxy for line-of-sight visibility to parks/greenspace from each façade sensor) were simulated and discretized into 2-5 ordered classes based on established standards (e.g., BRE and EN guidance) and study-defined VP thresholds. Geometric features describing sensor location, façade orientation, façade-edge distances, and outward obstruction distance were extracted. Five algorithms were compared across the six multi-class tasks; LightGBM achieved the best overall performance (mean Accuracy = 0.95; mean F1-macro = 0.92 on a 20% hold-out set, averaged across the six metrics) and maintained strong generalization on 20 independent layouts generated outside the main dataset. SHAP attribution highlighted façade orientation and obstruction/position features (e.g., sensor height and outward obstruction distance) as dominant predictors, whereas façade-edge distances showed relatively minor contributions. The proposed framework enables near-real-time façade heatmaps for daylight, solar exposure, and parks/greenspace visibility–related performance, supporting rapid screening and early-stage optimization of urban design options.
