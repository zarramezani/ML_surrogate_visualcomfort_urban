# ML_surrogate_visualcomdort_urban
In this repository, you will find the documents and data used in the research conducted by Zahra Ramezani, Zahra Sadat Zomorodian, Mohammad Tahsildoost

# Abstract

Visual comfort is a key dimension of environmental quality in dense urban blocks, yet multi-metric assessment via climate-based simulation is computationally demanding. This study develops a data-driven framework to predict façade-level performance classes for six visual comfort metrics using standard- and study-defined thresholds. Using Grasshopper with Honeybee/Ladybug, 2,000 parametric urban-block scenarios were generated for the climate of Tehran. For façade sensor grids, Vertical Sky Component (VSC), Vertical Daylight Factor (VDF), annual Sunlight Autonomy (SA), Direct Sun Hours (DSH), Peak Irradiance (Peak-Irr), and Visibility Percent (VP) were simulated and discretized into ordered classes, while geometric features describing sensor location, façade orientation, and outward obstruction distance were extracted. Five algorithms were compared for the six multi-class tasks; LightGBM achieved the best overall performance (mean Accuracy = 0.95; mean F1-macro = 0.92 on a 20% hold-out set) and maintained strong results on 20 independent layouts generated outside the main dataset. SHAP attribution highlighted façade orientation and obstruction/position features (e.g., sensor height and outward obstruction distance) as the dominant predictors, whereas façade-edge distances contributed less. The proposed framework enables interactive-speed façade heatmaps for daylight, solar exposure, and view-related performance, supporting rapid screening and early-stage optimization of urban design options.
